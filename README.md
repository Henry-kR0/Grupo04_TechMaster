¡Bienvenido al repositorio de **[Tienda en Electronica/MasterTech]**! Esta es una aplicación web moderna diseñada con una interfaz adaptable, optimizada con un modo oscuro nativo para reducir la fatiga visual y mejorar la experiencia del usuario.
**Problema principal**
  Esta pagina ha sido creada viendo la necesidad de muchas personas

## 🎨 Características Principales

- **Diseño Confortable:** Interfaz adaptada con una paleta de colores oscuros profundos (`#0a0e17` y `#111625`) que protegen la vista.
- **Totalmente Adaptable (Responsive):** Estructura basada en Grillas Nativas (`grid-2` y `grid-3`) y Media Queries que se ajustan automáticamente a computadoras, tablets y celulares.
- **Filtro Dinámico:** Barra de categorías interactiva controlada por JavaScript para segmentar productos o servicios en tiempo real sin recargar la página.
- **Formulario con Validación:** Formulario de contacto inteligente que valida campos vacíos y maneja notificaciones dinámicas (`¡SÍ FUNCIONA!`).

---

## 🛠️ Tecnologías Utilizadas

El proyecto fue desarrollado utilizando tecnologías web estándar de última generación:

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Estilos personalizados, variables globales (`:root`) y diseño adaptable.
- **JavaScript (ES6+):** Lógica interactiva, filtros de búsqueda y validación de formularios.
